LidySync Voltar para Home

Guia de Instalação e Uso

Aprenda de forma simples como instalar o LidySync no seu navegador e transforme qualquer player de vídeo em uma sala de cinema com seus amigos.

Pré-requisitos

🌐

Navegador Compatível

Você precisará estar usando o Google Chrome, Microsoft Edge, Opera ou Mozilla Firefox no computador.

Extensão Tampermonkey

O LidySync roda através deste gerenciador de scripts. Ele é seguro e gratuito.

Baixar Tampermonkey ↗

Passo a Passo de Instalação

1

Instale o Tampermonkey

Acesse a loja de extensões do seu navegador e adicione o Tampermonkey ou clique aqui. Após instalar, fixe-o na sua barra de tarefas para facilitar o acesso.

2

Crie um Novo Script

Clique no ícone do Tampermonkey na barra do seu navegador e selecione a opção "Adicionar novo script" (Create a new script).

3

Cole o Código do Loader

No editor de texto que abrir, apague todo o código padrão que já vem escrito lá e cole exatamente o código abaixo:

(async function () {
    'use strict';
    const SCRIPT_URL = "https://raw.githubusercontent.com/OFaceOff/LidySync/main/lidysync.user.js";
    async function loadScript(url) {
        return new Promise((resolve, reject) => {
            const s = document.createElement("script");
            s.src = url;
            s.onload = () => resolve();
            s.onerror = () => reject(new Error("Erro ao carregar: " + url));
            document.head.appendChild(s);
        });
    }
    async function loadFirebase() {
        if (window.firebase) return;
        const libs = [
            "https://www.gstatic.com/firebasejs/9.23.0/firebase-app-compat.js",
            "https://www.gstatic.com/firebasejs/9.23.0/firebase-firestore-compat.js",
            "https://www.gstatic.com/firebasejs/9.23.0/firebase-auth-compat.js"
        ];
        for (const lib of libs) { await loadScript(lib); }
    }
    async function fetchWithRetry(url, attempts = 3) {
        for (let i = 1; i <= attempts; i++) {
            try {
                const res = await fetch(url + "?t=" + Date.now(), { cache: "no-store" });
                if (!res.ok) throw new Error("Status: " + res.status);
                const text = await res.text();
                if (!text || text.length < 50) throw new Error("Script vazio");
                return text;
            } catch (err) {
                if (i === attempts) throw err;
                await new Promise(r => setTimeout(r, 1000 * i));
            }
        }
    }
    try {
        await loadFirebase();
        const code = await fetchWithRetry(SCRIPT_URL);
        (new Function(code))();
    } catch (err) { console.error("[LidySync] Erro no loader:", err); }
})();

Por que usar o Loader?

O Loader é um código inteligente. Com ele instalado, o LidySync irá baixar e atualizar automaticamente para a versão mais recente toda vez que houver uma novidade, sem que você precise fazer nada manualmente!

Precisa do código original ou quer ver as atualizações? Acesse o nosso GitHub Oficial.
4

Salve o Script

Após colar o código, pressione Ctrl + S no teclado ou vá em Arquivo > Salvar no menu superior do Tampermonkey.

5

Abra seu site de vídeo favorito!

Pronto! Acesse qualquer site compatível (YouTube, Netflix, Crunchyroll, etc) e o ícone do LidySync vai aparecer no canto inferior direito da sua tela.

Como Usar na Prática

Usar o app é ainda mais fácil do que instalar. Veja o fluxo básico:

1

Ao entrar em um site com vídeo, clique no botão do LidySync no canto direito.

2

Crie uma nova sala com nome e senha, ou insira os dados de uma sala existente.

3

No menu da sala, clique em "Compartilhar Sala" e envie o convite para seus amigos.

4

Use o chat para conversar. Quando o Anfitrião (quem criou a sala) der play ou pausar o vídeo, a sincronização acontecerá para todos automaticamente.

Principais Recursos

O LidySync está repleto de ferramentas pensadas para melhorar a sua experiência.

🖥️ Modo Teatro

Fixa o chat na lateral da tela e redimensiona o vídeo para que não haja sobreposição. Assistir e teclar ao mesmo tempo!

⏱️ Sincronização por Host

O anfitrião pode usar o Play Sincronizado para criar uma contagem regressiva de 3 segundos visual para todos da sala.

💬 Chat Completo

Envie imagens, busque GIFs, cite seus amigos com @ e compartilhe a sua câmera como foto rapidamente.

⏳ Timecodes Clicáveis

Envie o tempo do filme (ex: 01:25) e ele se torna um botão. Quem clicar, pula direto para aquela cena.

Dicas Úteis

  • Certifique-se de que todos estão exatamente na mesma página de vídeo antes de iniciar a sessão.
  • Utilize uma conexão de internet estável para que a sincronização não sofra com pequenos atrasos (delay).
  • O controle está nas mãos do Host! Apenas a pessoa que criou a sala afeta a linha do tempo dos outros convidados.
